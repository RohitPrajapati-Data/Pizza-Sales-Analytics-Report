# Pizza Sales Analytics Dashboard | Power BI

## Project Overview

This project presents an interactive **Pizza Sales Performance Dashboard** built using **Power BI**. The dashboard helps analyze pizza sales performance, revenue trends, order patterns, pizza category performance, pizza size distribution, and best/worst-selling pizzas.

The main objective of this project is to help business users understand sales performance and make better decisions related to product demand, sales trends, and customer ordering behavior.

## Dashboard Preview

[View PDF Report](Pizza_Sales_Report.pdf)

## Key KPIs
* Total Revenue: 817.62K
* Total Orders: 21,334
* Average Order Value: 38.32
* Total Pizzas Sold: 49,559
* Average Pizzas per Order: 2.32

## Dashboard Insights
* The Thai Chicken Pizza generated the highest revenue.
* The Classic Deluxe Pizza received the highest number of orders.
* The Classic Deluxe Pizza was also the top pizza by quantity sold.
* The Brie Carre Pizza performed the lowest in revenue, orders, and quantity sold.
* Large-size pizzas contributed the highest sales share.
* Classic category pizzas had the highest sales distribution.
* Thursday recorded the highest number of orders.
* Saturday recorded the lowest number of orders.
* Monthly revenue remained mostly stable, with some months showing higher sales performance.
